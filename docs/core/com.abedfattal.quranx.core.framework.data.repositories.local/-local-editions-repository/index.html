class LocalEditionsRepository
Content copied to clipboard

Use LocalEditionsRepository to save and apply local queries on Edition table. To save Edition you should call the one of the methods: addEdition or addAllEdition. then you can performs local queries on Table, otherwise you'll get a nullable or empty list, (obviously) since the requested Edition does not exist in the database.
