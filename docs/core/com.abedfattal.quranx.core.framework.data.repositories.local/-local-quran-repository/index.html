LocalQuranRepository

class LocalQuranRepository
Content copied to clipboard

Use LocalQuranRepository to save and apply local queries on Aya table . To save Aya you should call the appropriate methods, such as LocalBasedQuranRepository.downloadQuranBook, addSurah, or addAyat, etc... then you can performs local queries on Table, otherwise you'll get a nullable or empty list, (obviously) since the requested Aya does not exist in the database.
