core

A is the core that provides Quran data sources based on AlQuran.cloud with stylish Kotlin architecture out of box, which provide data such as Quran, Tafseer, translation, transliteration, word by word, etc... In the core there are three options data sources:

  • Local data source, which performs local query to provides the data, see LocalDataSource.

  • Remote data source, which performs queries on remote server to provides the data, see RemoteDataSource.

  • Local-based data source, which performs queries on remote only if the data is not available in the local source, see LocalBasedDataSource.

Note that you could rely fully on the remote data source, and safely ignore the local and the local-based data sources, for more information see the package documentation.
