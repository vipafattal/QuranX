ProcessState

sealed class ProcessState<T>
Content copied to clipboard

A wrapper class for processes that can Loading, Success, or Failed. This class used by the library to simplify requesting data code structure (rather then using try-catch pattern) which really improves the code readability. e.g. requesting data from a remote API service may fail, success, or even waiting for response.

The type T represents the data type of the current process.
