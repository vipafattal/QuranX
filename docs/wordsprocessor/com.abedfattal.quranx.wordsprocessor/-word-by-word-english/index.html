WordByWordEnglish

object WordByWordEnglish
Content copied to clipboard

WordByWordEnglish use to extract Arabic Quran words meaning in English. It only supports the following text editions quran-wordbyword, quran-wordbyword-2. It's recommend to use getWordOfAyaV2 method which is based on quran-wordbyword-2 for better results and also you get transliteration for out the box.
