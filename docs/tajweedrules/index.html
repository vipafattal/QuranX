tajweedrules

A library for listing Tajweed rules in words of Quran verses which is based on quran-tajweed edition of alquran.cloud.

Setup

https://github.com/vipafattal/QuranX/blob/master/README.md#instalition

Usage

Tajweed Rules depended on Tajweed to create a custom action on words that list all rules in each word of the verse, so first make sure you have Tajweed Processor in your app, Then pass your Tajweed object into TajweedRules:

 val tajweed: Tajweed = Tajweed()
 val tajweedRules = TajweedRules(tajweed)

Once you have TajweedRules, you could call:

  val tajweed: Tajweed = Tajweed()
  val tajweedRules = TajweedRules(tajweed)

  val rules : List<WordsWithRules> =  tajweedRules.getRulesOfAya(ayaText)

You can also list all supported rules see docs.
